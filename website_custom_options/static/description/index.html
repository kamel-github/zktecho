Make The Products Fully Customizable For Your Customers
Now, use customized options rather than product variants in Odoo Website. So, you can give ample options to your customers without any hassle. The multiple options allow your customers to customize the products as they want directly from Odoo Website. You can create customizable products using the Custom Options. The best part is that custom options work just like variants, however, you don't need to manage stocks unlike variants.
Offer Multiple Input Options to Your Customers
Set appropriate input methods such as the dropdown menu, radio button, enter time, date text or upload a file to add a custom option in an order line. So that the customers can customize the products in the way they want.
Let Customers' pick the Custom Options Directly From Odoo Website
The module allows you to offer the customizable products in Odoo Website. Your customers need to simply pick the options and customize the product in the way they want.
Understand Odoo Website Custom Options With an Example
Suppose, you want to run a bakery online; then creating variants for all possible customization is not feasible. In this case, you can create custom options in place of attributes and let your customers decide what needs to be prepared. Set unique inputs for different products like dropdown, multi-select, checkboxes and even file upload for a perfectly customized customer experience.
